# Simple ToDo
Simple program for create todo list using Rust

```bash

## build the binary
cargo build --bin todo

## Add the task
./target/debug/todo add "Task 1"

## Complete the task
./target/debug/todo complete "Task 1"

## Update status the task
./target/debug/todo pending "Task 1"

```
